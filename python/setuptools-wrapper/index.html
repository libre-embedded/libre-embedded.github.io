setuptools-wrapper (0.2.11)#

A simple interface to setuptools’s setup function.

Consider sponsoring development.

(interface documentation)

Python Version Support#

This package is tested with the following Python minor versions:

  • python3.13

  • python3.14

Platform Support#

This package is tested on the following platforms:

  • ubuntu-latest

  • macos-latest

  • windows-latest

Introduction#

That’s it. I personally think there are way too many configuration options when “packaging” software and I want to homogenize my own packages. Thus: setuptools-wrapper.
